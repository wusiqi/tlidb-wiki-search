# Wiki Search — 需求文档

## 数据需求

### 1. 游戏机制
- 机制名字
- 机制描述

### 2. 英雄职业特性
- 英雄名字
- 英雄各个特性技能名
- 特性技能效果描述（含需求等级）

### 3. 天赋
- 天赋类别（巨力之神、知识之神、狩猎之神、征战之神、欺诈之神、机械之神）
- 核心天赋（名字 + 效果）
- 小型天赋 / 中型天赋（效果 + 所属神系）

### 4. 传奇装备
- 传奇装备名字
- 传奇装备词缀属性
- 装备类型标签（头盔/项链/单手剑/双手锤等，从图片路径自动识别）

### 5. 棱镜
- 棱镜名字
- 棱镜效果
- 棱镜可出现词缀

### 6. 契灵
- 契灵名字
- 契灵效果

### 7. 命运
- 命运名字
- 命运效果

### 8. 技能
- 技能类别（主动/辅助/被动/触媒/触发/崇高/华丽）
- 技能名字
- 技能效果描述
- 技能标签（法术/范围/持续/火焰/物理等）

### 9. 打造（统一词缀来源）
- 词缀效果
- 来源（装备类型：法杖/项链/戒指等）
- 类型（词缀等级：至臻前缀/至臻后缀/侵蚀基底等）

## 搜索功能需求

- 多关键词 AND 搜索（Enter 添加，✕ 移除，Backspace 删除最后一个）
- 在所有字段上匹配（名字、描述、标签、来源、子类型）
- 搜索结果按类别分组展示
- 搜索结果去重和合并

## 前端功能需求

### 分类过滤（左侧 Sidebar）
- 显示各类别及匹配数量
- 首次点击 = 单选（只选中该类别，取消其他）
- 之后点击 = 多选 toggle

### 常用标签面板（右侧 TagPanel）
- 可折叠分组：元素、技能类型、异常状态、装备类型、天赋、英雄
- 点击标签 = 添加/移除搜索关键词
- 标签与数据中的 tags 对应

### 结果展示
- 打造：三列表格（词缀效果 | 来源 | 类型）
- 英雄特性：嵌套布局（英雄名下展示各特性技能）
- 传奇装备：显示装备类型标签 + 多行词缀（按 `｜` 分行）
- 其他类别：卡片布局（蓝色名字 + 灰色描述）
- 关键词橙色高亮

### 响应式布局
- 桌面（≥1024px）：三栏（Sidebar + 结果 + TagPanel）
- 平板（768-1023px）：两栏（Sidebar + 结果，TagPanel 隐藏）
- 手机（<768px）：单栏（Sidebar 变为横向滚动 pills，TagPanel 隐藏）

## 部署需求

- 构建时生成数据（`build-data.ts`），运行时不访问 wiki
- Vercel 部署，GitHub 连接自动部署
- 自定义域名 `www.wsiq.cloud`
- 中国用户可通过自定义域名访问（`.vercel.app` 在中国被屏蔽）
